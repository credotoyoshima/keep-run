評価機能の特徴

多様な表示形式

  • • カレンダービューで月間俯瞰
  • • ヒートマップで傾向を可視化
  • • 統計データで成長を確認
  • • TOP DAYSで成功体験を振り返り

シンプルな記録

  • • 5段階の星評価
  • • オプションのコメント機能
  • • クイック選択ボタン
  • • 21時に記録を促すリマインド

成長の可視化

  • • 月間平均スコアの推移
  • • 連続記録日数のカウント
  • • 達成率の表示
  • • インサイトによる気づき

UI/UXのポイント

評価機能は、日々の振り返りを習慣化し、長期的な成長を可視化することを目的としています。 カレンダーとヒートマップの2つのビューを切り替えることで、 詳細な日々の記録と俯瞰的な傾向の両方を確認できます。 シンプルな5段階評価により、記録のハードルを下げています。

デザインの特徴

統計表示には極細フォント（28px, font-weight: 200）を使用し、データを美しく表現。 カレンダーの各日付には小さな星で評価を表示し、一目で月間の傾向を把握可能。 ヒートマップは濃淡のグレースケールで、GitHubの貢献グラフのような視覚的なフィードバックを提供します。
